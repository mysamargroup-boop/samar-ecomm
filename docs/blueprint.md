# **App Name**: CommerceWave

## Core Features:

- Product Management: Admin interface for creating, updating, and deleting products, including price, variants, inventory, and images using server actions.
- Category Management: Admin interface for managing categories, including creating, updating, and deleting categories, also using server actions.
- Order Management: Admin interface to view and update order statuses.
- WhatsApp OTP Login: Admin login using WhatsApp OTP verification flow.
- Stripe Checkout: Stripe checkout integration with webhook verification to handle payments and update inventory using server actions.
- Dynamic Category Pages: Display products based on categories with SEO optimization.
- AI Product Description Generation: An AI tool which helps generate product descriptions for new products to improve searchability.
- Homepage Slider: Image slider on the homepage showcasing featured products or promotions.
- Product Share Button: Share button on product pages to allow users to share products on social media.
- Footer with Essential Pages: Footer section with links to Terms and Conditions, Privacy Policy, Shipping Information, and Contact Us pages.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and stability.
- Background color: Light Gray (#F0F2F5) for a clean and modern feel.
- Accent color: Amber (#FFC107) for highlighting key actions and elements.
- Body font: 'Manrope' sans-serif for a clean and modern feel suitable for body text.
- Headline font: 'Manrope' sans-serif for impactful headlines and titles.
- Use modern, minimalist icons from a library like 'Lucide React' for a consistent and professional look.
- Use a grid-based layout with consistent spacing to create a balanced and responsive design.
- Implement subtle animations for transitions and interactions to enhance user experience.
